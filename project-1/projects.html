T-Rex

The Chrome DinoGame, also known as the offline T-Rex game, was created by Google Chrome engineer, Sebastien Gabriel, in 2014. The game was designed as a fun way to entertain users when their internet connection was lost. The game features a pixelated T-Rex that jumps over cacti and avoids obstacles, with the player earning points for every successful jump.

In my implementation, I used Java Swing to create a cross-platform Java application that emulates the classic gameplay experience on Mac, Ubuntu, and Windows.

VEX Robotics

VEX Robotics is an educational robotics platform that allows students to design, build, and program robots. The robots are constructed using modular components, including motors, sensors, and structural elements, that are assembled using metal brackets and screws. The robots are programmed using the RobotC language, a C-based language specifically designed for use in robotics. The RobotC language allows students to write code that controls the robot's movements, responds to sensor inputs, and makes decisions based on logical statements. It is an easy-to-learn language that has a robust set of libraries and functions that are specific to VEX Robotics.

Some of the main components of VEX Robotics include the VEX Cortex Microcontroller, VEX motors, and VEX sensors. The Cortex Microcontroller is the brain of the robot and controls all of its functions. The motors provide the robot with power and control its movement, while the sensors provide feedback on the robot's environment and allow it to respond to its surroundings.

Spinning Structs

"Donut.c" is a computer program that generates a 3D image of a torus (a donut shape) using ASCII characters. It was originally created by computer programmer and mathematician John C. Beatty in the mid-1980s, but gained popularity when it was later modified and optimized by another programmer, Andy Sloane, in 2006.

The program is written in C and uses trigonometric functions to calculate the coordinates of each character on the screen, creating the illusion of a 3D object. The program's popularity is due to its elegant and concise code, which demonstrates the power of mathematical algorithms in computer graphics.

In my implementation, I used embedded matrices in a cube to simulate the floating illusion of "Donut.c". While my work was inspired by the online community, I am currently in the process of optimizing and thoroughly checking for errors to enhance its completeness.

Current Project: DCFA

Discounted Cash Flow Analysis (DCFA) is a financial analysis method used to evaluate the value of an investment based on its projected future cash flows. DCF analysis involves estimating the future cash flows of an investment and discounting them back to their present value using a discount rate, typically the cost of capital. The resulting present value represents the estimated intrinsic value of the investment.

I've been expanding my skill set in mobile application development and decided to apply my knowledge to iOS and Android platforms. Currently, I'm working on implementing DCF analysis in Swift for iOS devices.
